# SpaceShip-Game
A little game about a spaceship trying to survive in space :)

Basically, in this simple game you are the spaceship and you're trying to navigate through space for the most amount of time possible. You are equipped with a laser in order to blast any meteors that come towards your way (you can also dodge if you have ninja-like reflexes). The game continues onto infinity, just like space, so you will never be bored.

This was essentially done using python and I tested it with the help of the pygame-ce version. Then, I had to make everything web deployable through the usage of pygbag! I was able to learn a lot through this game by learning collisions, and also making the continous motion of the stars and meteors. This was a really good learning experience and I'm very happy to finish this little game!

Tech Stack: Python, Pygame-CE, Pygbag

AI Used: None

# Features
- 3 modes of the game
  - Keyboard mode (where you play with your keyboard of course)
  - Mouse mode (where you play with your mouse, who would of thought?)
  - Target mode (maybe a fun little twist I shall say)
- Menu in order to choose one of the 3 modes in this game
- Sounds implemented in this game from background music to collisions
- Moving objects to give off the illusion of going upwards
- A timer that will time how many seconds you've been alive each time
- Hit counter for the total amount of meteors you've hit each run
- Also a simple collision effect when your laser destroys that meteor or you get destroyed by a meteor
